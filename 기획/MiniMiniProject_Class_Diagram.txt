## Naming 작업
클래스 다이어 그램(Class Diagram)


// Ex ] sakeAry[][] : sakeAry{"고유번호", "이름", "설명", "브랜드"};

// Ex ] priceAry[][] : priceAry{{1000, 2000, 3000, 4000}  - sake price
//			        {2000, 5000, 7000, 2400}}; - soju price
** AlcoholDB : 술정보 DB
	- sakeAry[][] : String		- fruitWineAry[][] : String
	- sojuAry[][] : String 		- priceAry[][] : int
	
// Ex ] RegisterAry[][] : RegisterAry{"회원 id", "회원 pwd", "회원 이름", "회원 생년월일"}; ## 물어봐야 될듯 ## => 처음 초기화 작업
** RegisterDB : 회원 정보 DB
	- RegisterAry[][] : String

** AlcoholDTO(Data Transfer Object) : 술정보
	- number : String		- brand : String
	- name : String			- explanation : String 
	- price : int

** SignInUpDTO : 로그인, 회원가입
	로그인			회원가입
	- id : String		- name : String
	- password : String	- birth : String
				- passwordConfirmation : String
	
	+ getter / setter

** AlcoholDAO(Data Access Object)
	- userAPI : UserAPI	- alcoholdb : AlcoholDB
	- scanner : Scanner	- registerdb : RegisterDB
	
	+ printMenuInfo() : void
	+ dtoSet() : void
	+ custChk(String readCust) : int
	+ custInfo() : void -> Sake, Soju, FruitWine, SignIn, SignUp

** AlcoholMain
	+ main(args String[])

** Sake - 사케(日本酒)
	- alcoholDTO : AlcoholDTO => dto 객체  
	- scanner : Scanner

	+ printSakeMenu(): void : sakeAryDB에 저장된 제품 목록 출력.
	+ SakeInfoNumber(String scanner) : int => 사용자가 입력한 값의 유효성 검사 후 number 값에 전달.

** Soju - 소주(焼酎)
	- alcoholDTO : AlcoholDTO => dto 객체
	- scanner : Scanner

	+ getSojuInfoNumber(String scanIn) : int  // 유효성 검사.
	+ SojuInfoByNumber(int number): void // 유효성 검사가 끝난 number로 상품정보표시
	+ SojuMenuprint() : void

** FruitWine - 과실주(果実酒)
	- alcoholDTO : AlcoholDTO => dto 객체
	+ scanner     :  Scanner

	+ printFruitWIne :  void  : 과실주 메뉴 출력
	+ selectfruitWine :  void  : 과실주 메뉴 항목 입력에 따른 출력


** SignIn - 로그인
	- signInUpDTO : SignInUpDTO : 로그인 회원가입 dto 객체
	- scanner : Scanner

	+ signChk() : string  => ID, pw 동일 여부 체크(db에서 정보 수신)
	+ signInMenuPrint() : void => 로그인 화면 출력

** SignUp - 회원가입
	- signInUpDTO  : SignInUpDTO  : 로그인 회원가입 dto 객체
	- scanner : Scanner
 
	+ putSignup() : string[] => 회원 정보 보내기
	+ signUpMenuPrint() : void => 회원가입 화면 출력
